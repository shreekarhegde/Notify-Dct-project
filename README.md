# Notify-Dct-project
# Overview
A platform which connects employees within their working environment. This enables users to create events, invite them, create calendar, send messages and set reminders.
# Dependencies
1. cors
2. express
3. lodash
4. mongoose
5. nodemon
6. socket.io
7. strftime
